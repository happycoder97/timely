# Timely
Kanban board, Timetable, Weekly scheduler. Built using Electron and Vue and TailwindCSS.

Currently in ~alpha~ dog fooding phase.
Issues and PRs welcome.

## Run
```
cd timely
yarn install
yarn dev
```
